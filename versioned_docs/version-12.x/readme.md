# Laravel 문서 (Laravel Documentation)

Laravel 문서의 온라인 버전은 [https://laravel.com/docs](https://laravel.com/docs)에서 확인할 수 있습니다.

## 기여 가이드라인 (Contribution Guidelines)

**현재 안정 버전**에 대한 문서를 제출하는 경우에는 해당 버전의 브랜치에 제출해야 합니다. 예를 들어 Laravel 12 문서는 `12.x` 브랜치에 제출합니다. 다음 릴리스를 대상으로 하는 문서는 `master` 브랜치에 제출해야 합니다.
